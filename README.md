# CeiloCraft React App

Clean JavaScript React app refactored from Lovable-generated TypeScript output.

## Stack

- React 18
- Vite 5
- React Router
- Tailwind CSS
- Framer Motion
- Radix UI primitives (only the components used by the app)

## Scripts

```bash
npm run dev
npm run build
npm run preview
npm run lint
npm run test
```

## Project Structure

```text
src/
  assets/
  components/
    ui/
  hooks/
  i18n/
  lib/
  pages/
  App.jsx
  main.jsx
```

## Notes

- TypeScript and TS configs were removed.
- Unused Lovable/shadcn generated files were removed.
- ESLint and Vitest are configured for JavaScript/JSX.
